## 🎯 O que é?

Um conversor de moedas bem direto. Você coloca um valor, escolhe as moedas e pronto - ele converte pra você. Sem complicações, só JavaScript puro mesmo.

## ✨ Funcionalidades

- ✅ Converte entre 4 moedas principais
- ✅ Interface limpa e intuitiva
- ✅ Totalmente responsivo (mobile, tablet, desktop)
- ✅ Design moderno com gradientes
- ✅ Conversão em tempo real

## 🚀 Como usar

1. Clone o repositório
```bash
git clone https://github.com/seu-usuario/money-exchange.git
```

2. Abra o arquivo `index.html` no navegador

3. Escolha a moeda que você tem e a que quer

4. Digite o valor e clique em "Converter Agora"

5. Vê o resultado na hora ✨

## 📊 Taxas de Câmbio

Usando valores fixos por enquanto:

- 1 USD = R$ 4.98
- 1 EUR = R$ 5.86
- 1 GBP = R$ 6.73

Depois vou integrar uma API real pra taxas atualizadas.
