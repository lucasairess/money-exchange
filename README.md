# 💱 Money Exchange

Conversor de moedas que não parece ter sido feito por um robô.

## O que é isso?

Um conversor de moedas simples mas funcional. Você coloca um valor, escolhe as moedas e pronto - ele converte pra você. Sem complicações, sem APIs malucas, só JavaScript puro mesmo.

## Como funciona

1. Clone/baixe o projeto
2. Abre o `index.html` 
3. Escolhe a moeda que você tem e a que quer
4. Digita quanto você quer converter
5. Clica no botão e vê a mágica acontecer ✨

## Stack

- **HTML** - A estrutura básica
- **CSS** - Deixei bonito e responsivo
- **JavaScript** - A lógica toda aqui

## Moedas suportadas

Real, Dólar, Euro e Libra. Se precisar de mais, é só adicionar.

## Taxas de câmbio

Tá usando valores fixos por enquanto:
- 1 USD = R$ 4.98
- 1 EUR = R$ 5.86
- 1 GBP = R$ 6.73

Depois integro uma API real pra isso.
